# n8n & Automation-Dienstleistungen verkaufen – Strategie

## Warum n8n / Automation so ein gutes Geschäftsfeld ist

- **Hohe Nachfrage**: Jedes Unternehmen hat Prozesse die automatisiert werden können
- **Wenig Konkurrenz**: Noch relativ neue Nische in Deutschland
- **Hohe Wertschöpfung**: Du sparst Kunden Zeit → sie zahlen gerne dafür
- **Retainer-freundlich**: Setup einmalig + monatliche Betreuung = wiederkehrendes Einkommen
- **Skalierbar**: Templates und Workflows können wiederverwendet werden

## Welche Unternehmen brauchen das?

### Perfekte Zielkunden für n8n:
- **Kleine E-Commerce-Shops** (Bestellbestätigungen, Lagerwarnung, Kundennachrichten)

- **Agenturen** (Report-Generierung, Client-Updates)
- **Gastronomie** (Reservierungsmanagement)
- **Handwerksbetriebe** (Angebotsprozesse, Rechnungsstellung)
- **Coaches/Berater** (Onboarding-Prozesse, Follow-Up)

### Wie du sie findest:
1. Lokale Unternehmen in NRW (Google Maps, Yelp)
2. LinkedIn-Suche nach "Geschäftsführer" + Branche + Stadt

## Automatisierungen die am häufigsten gebucht werden

### Einsteiger-Automationen (gut für Referenzen):
1. **Lead-Capture** → CRM: Kontaktformular-Eintragung landet automatisch im CRM (HubSpot, Pipedrive etc.)
2. **Terminbuchung** → E-Mail + Kalender: Calendly-Buchung erstellt Google Calendar-Eintrag + sendet Bestätigung
3. **Rechnungs-Workflow**: Neue Rechnung in Lexoffice → PDF als E-Mail an Kunde

### Mittlere Automationen:
4. **E-Commerce-Workflow**: Neue Bestellung → Lager prüfen → Versand einleiten → Kunde informieren
5. **Social-Media-Posting**: Ein Post erstellen → automatisch auf Instagram + LinkedIn + Facebook
6. **Newsletter**: Neuer Blog-Artikel → automatisch Newsletter versenden

### Fortgeschrittene Automationen:
7. **KI-gestützte Kundenanfragen**: Kundenanfrage per E-Mail → GPT-Analyse → vorgefertigte Antwort
8. **Reporting**: Monatlich automatischer Umsatzbericht aus verschiedenen Quellen

## Preisfindung für Automationen

### Statt Zeit zu verkaufen → Wert verkaufen!

Schlechte Kommunikation: "Ich baue einen n8n-Workflow, kostet 3 Stunden à 60 €"
Gute Kommunikation: "Diese Automation spart Ihnen ca. 5 Stunden Arbeit pro Woche. Bei einem Stundenlohn Ihres Mitarbeiters von 20 € = 400 €/Monat Ersparnis. Mein Setup kostet 600 €."

→ Der Kunde zahlt 600 € und spart 4.800 €/Jahr. Klares Argument!

### Pricing-Modell:

```
Setup-Gebühr:    Einmalig für den Aufbau des Workflows
                 (je nach Komplexität: 150-4.000 €)

Wartungs-Retainer: Monatlich für:
                   - Überwachung ob die Automation läuft
                   - Kleine Anpassungen
                   - Neue Trigger oder Steps hinzufügen
                   (50-200 €/Monat)
```

## Coaching verkaufen – Rechtliches und Strategie

### Darf ich "Coach" sein ohne Ausbildung?

**JA!** Der Begriff "Coach" ist in Deutschland nicht geschützt.
Du brauchst keine formale Coach-Ausbildung um Coaching anzubieten.

**Was du NICHT darfst:**
- Geschützte Berufsbezeichnungen: "Psychotherapeut", "Psychologe", "Arzt"
- Medizinische oder psychotherapeutische Beratung
- Rechtsberatung (nur Anwälte!)
- Steuerberatung (nur Steuerberater!)

**Was du DARFST:**
- IT-Coaching, Business-Coaching, Medien-Coaching
- Technische Schulungen (n8n, Web etc.)
- Prozessberatung
- Soft-Skills (als Coach ohne Schutzbezeichnung)

### Dein Angebot schärfen:
Statt allgemeines "Coaching" → **Spezifisch werden:**
- "n8n Automation Einführungskurs für Unternehmen"
- "Workshop: Digitalisiere deinen Unternehmensprozess in einem Tag"
- "Webdesign für Einsteiger – 1:1 Coaching"

## n8n Self-Hosting für Kunden einrichten – Einnahmenquelle

Viele Unternehmen wollen n8n auf eigenem Server.

**Service:** Du richtest ein:
1. Hetzner Cloud Server (5-10 €/Monat)
2. Docker + n8n Installation
3. Backups einrichten
4. Kunden einweisen

**Was du berechnest:**
- Setup einmalig: 200-400 €
- Monatliche Server-Betreuung: 50-100 €/Monat
- Du verwaltest den Server → Kunde hat keinen Aufwand

**Wichtig:** Dann bist du für den Server mitverantwortlich → Berufshaftpflicht noch wichtiger!
