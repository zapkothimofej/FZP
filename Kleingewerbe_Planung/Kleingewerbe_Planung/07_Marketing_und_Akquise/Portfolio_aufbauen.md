# Portfolio aufbauen – Von Null zur überzeugenden Referenz

---

## Warum das Portfolio das Wichtigste ist

Kunden kaufen nicht dich – sie kaufen das Vertrauen dass du liefern kannst.
Ein gutes Portfolio gibt dieses Vertrauen.

**Ohne Portfolio:** "Ich mache gute Videos" (unbeweisbar)
**Mit Portfolio:** "Schau hier, das habe ich für Kunde X gemacht" (beweisbar)

---

## Schritt 1: Sofort starten – auch ohne Aufträge

Du musst nicht auf echte Kunden warten!

**Eigeninitiativ-Projekte:**

Für Video/Foto:
- Lokales Café oder Restaurant → bitte um kostenloses Shooting für Portfolio
- Dein eigenes "Imagevideo" (zeige deinen Arbeitsbereich, dein Workflow)
- Freunde mit kleinem Business (Friseur, Fitnesstrainer etc.)
- Nachher/Vorher-Retusche an öffentlichem Bildmaterial

Für Web / n8n:
- Bau deine eigene Portfolio-Website → Das IST schon Portfolio!
- Erstelle einen Demo-n8n-Workflow (Screenshot + Beschreibung)
- Bau eine Demo-Website für ein fiktives lokales Unternehmen

## Portfolio-Website – Muss-haves

### Struktur:
```
Startseite
    ↓
Leistungen (kurz, klar)
    ↓
Portfolio / Arbeiten
    ↓
Über mich (kurz, persönlich)
    ↓
Kontakt (Formular + direkte E-Mail)
```

### Was jedes Portfolio-Projekt zeigen muss:
1. **Das Ergebnis** (Foto, Video, Screenshot der Website)
2. **Die Aufgabe** (Was wollte der Kunde?)
3. **Die Lösung** (Was hast du gemacht?)
4. **Den Effekt** (Was hat es gebracht? Mehr Kunden, bessere Online-Präsenz etc.)

**Beispiel:**
```
PROJEKT: Imagefilm für Mustermann Sanitär GmbH
─────────────────────────────────────────────
Aufgabe: Das Team sollte sympathisch und professionell
         für die neue Website dargestellt werden.
Leistung: 2-Tages-Drehtag, 90-Sekunden-Imagefilm, 3 Testimonial-Videos
Ergebnis: "Der Film läuft auf unserer Website und wir bekommen
          deutlich mehr Anfragen." – Inhaber Hans Mustermann
─────────────────────────────────────────────
[VIDEO EMBED / THUMBNAIL]
```

## Testimonials einsammeln – So geht's

Nach jedem abgeschlossenen Projekt:

**E-Mail-Vorlage:**
```
Betreff: Kurze Bitte – Feedback zu unserem Projekt

Hallo [Name],

ich hoffe, Sie sind mit dem Ergebnis unserer Zusammenarbeit
zufrieden!

Würden Sie mir einen kurzen Satz (2-3 Zeilen) schreiben den
ich als Referenz auf meiner Website und in meinen Angeboten
verwenden darf?

Zum Beispiel:
- Was hat die Zusammenarbeit für Sie gebracht?
- Was hat Ihnen besonders gut gefallen?

Das wäre super hilfreich für mich!

Beste Grüße,
[Dein Name]
```

## Plattformen wo du dein Portfolio zeigst

| Plattform | Für was | Aufwand |
|---|---|---|
| **Eigene Website** | Alles – Hauptanlaufstelle | Einmal aufbauen |
| **Behance** | Design, Video, Foto | Portfolio hochladen |
| **Instagram** | Foto, Video | Täglich posten |
| **YouTube** | Videos | Videos hochladen |
| **LinkedIn** | B2B, IT, Coaching | Projekte als Artikel |
| **GitHub** | Code, Web-Projekte | Repositories zeigen |
| **Vimeo** | Professionelle Videos | Nur Videos |

## Was du mit deinen ersten Projekten tun solltest

```
Referenzprojekt abgeschlossen:
    ↓
Fotos / Screenshots / Video machen
    ↓
Fallstudie schreiben (Aufgabe → Lösung → Ergebnis)
    ↓
Testimonial einholen
    ↓
Auf Website hochladen
    ↓
Auf Instagram / LinkedIn posten ("Neues Projekt fertig!")
    ↓
In Angeboten verlinken ("Hier ein ähnliches Projekt: [Link]")
```

## Portfolio für n8n – Besonderheiten

n8n-Workflows kann man nicht immer öffentlich zeigen (Kundendaten!).

**Alternativen:**
1. **Anonymisierter Screenshot**: Namen/Daten unkenntlich machen
2. **Demo-Workflow**: Gleichen Workflow mit Dummy-Daten nachbauen und zeigen
3. **Beschreibung**: "Ich habe für ein Sanitärunternehmen einen Workflow gebaut der..."
4. **Video-Walkthrough**: Zeige wie der Workflow funktioniert (ohne echte Daten)

**Schütze immer Kundendaten!** Keine echten Firmennamen, E-Mails, Zahlen zeigen ohne Erlaubnis.
