## Computer / Schnitt-Workstation

### Minimum (für Video bis 4K):
| | Anforderung |
|---|---|
| CPU | Intel i7 / AMD Ryzen 7 (min. 8 Kerne) |
| RAM | 32 GB (besser 64 GB für 4K) |
| Grafikkarte | NVIDIA RTX 3060 oder besser |
| Speicher | 1TB NVMe SSD (System) + 4TB HDD (Archiv) |
| Monitor | 4K, farbtreu (Dell U2723D o.ä.) |
| **Gesamtkosten ca.** | **1.500-3.000 €** |

### Mac vs. Windows für Video/Foto:
| | Mac (M-Chip) | Windows |
|---|---|---|
| Performance | Sehr gut, effizient | Sehr gut |
| Kosten | Teurer (~1.800-3.500 €) | Günstiger (~1.000-2.000 €) |
| Software | Finale Cut Pro (Mac only) | Mehr Auswahl |
| ProRes | Native Unterstützung | Benötigt Plugin |
| Empfehlung | MacBook Pro M3 für unterwegs | Desktop für Studioarbeit |

## Was du steuerlich absetzen kannst

Alle Geräte sind **Betriebsausgaben!**

- Unter 800 € netto: Sofort im Kaufjahr abziehen
- Über 800 € netto: Über Nutzungsdauer abschreiben (AfA)

**AfA-Tabelle Equipment:**
| Gerät | Nutzungsdauer | AfA/Jahr (1.500 € Kauf) |
|---|---|---|
| Kamera | 7 Jahre | 214 €/Jahr |
| Laptop | 3 Jahre | 500 €/Jahr |
| Drohne | 5 Jahre | 300 €/Jahr |
| Studioausstattung | 5-10 Jahre | je nach Wert |

**Tipp:** Große Anschaffungen am Ende des Jahres kaufen → mindert den Jahresgewinn!

## Gebraucht kaufen – Wo?

- **eBay Kleinanzeigen**: Gut für Stative, Licht, ältere Kameras
- **MPB.com**: Spezialisiert auf Foto/Video, geprüft mit Garantie
- **Calumet Gebraucht**: Seriöser Fachhändler
- **Facebook Marketplace**: Lokal, günstig, kein Versandrisiko
- **Reddit r/photomarket**: International, Nischen-Equipment

**Immer prüfen:**
- Verschlussauslösungen bei Kameras (unter 50.000 = gut)
- Kratzer auf Objektiven prüfen (gegen Licht halten)
- Sensor auf Staub / Hotpixel testen
