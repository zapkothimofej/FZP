# Kunden-Onboarding – Checkliste für jeden neuen Auftrag

*Für jeden neuen Kunden von Anfang an dieselbe Routine = weniger Fehler, mehr Professionalität*

---

## Phase 1: Erstgespräch / Anfrage

```
[ ] Anfrage erhalten (E-Mail / Social Media / Telefon)
[ ] Termin für Erstgespräch vereinbaren (Calendly oder manuell)
[ ] Vor dem Gespräch: Kunde googeln, Website anschauen
[ ] Gesprächsnotizen vorbereiten:
    - Was braucht der Kunde genau?
    - Was ist das Budget?
    - Was ist der Zeitrahmen?
    - Was ist das Ziel des Projekts?
[ ] Erstgespräch durchführen (mind. 30 Min.)
[ ] Notizen machen während des Gesprächs
```

---

## Phase 2: Angebot erstellen

```
[ ] Angebot schreiben (Vorlage in Ordner 10!)
[ ] Klar definieren was ENTHALTEN ist
[ ] Klar definieren was NICHT enthalten ist
[ ] Anzahlung einplanen (30-50% bei Projekten über 300 €)
[ ] Angebot als PDF per E-Mail schicken
[ ] Gültigkeitsdauer angeben (14-30 Tage)
[ ] Nachfassen wenn keine Antwort nach 5 Tagen
```

---

## Phase 3: Auftragserteilung

```
[ ] Schriftliche Bestätigung des Kunden erhalten
    (E-Mail-Antwort "Ich bestätige Angebot A-2025-XXX" reicht!)
[ ] Auftragsbestätigung / Vertrag schicken (Vorlage Ordner 10)
[ ] Anzahlungsrechnung stellen
[ ] Anzahlung abwarten BEVOR du anfängst!
[ ] Zugänge / Materialien vom Kunden anfordern:
    - Website-Zugänge, Hosting-Daten
    - Logos, Bilder, Texte
    - Briefing, Moodboard
    - Ansprechpartner klären
[ ] Projektordner anlegen (lokal + Cloud-Backup)
[ ] Termin im Kalender eintragen
```

---

## Phase 4: Projektdurchführung

```
[ ] Regelmäßige kurze Updates an den Kunden (~1x pro Woche)
[ ] Zwischenergebnisse zur Freigabe schicken (nicht einfach weiterarbeiten!)
[ ] Änderungswünsche schriftlich bestätigen lassen
[ ] Auf Scope Creep achten – extra Arbeit = extra Rechnung!
[ ] Alle Kundenkommunikation per E-Mail (Beweise!)
[ ] Deadlines einhalten oder früh kommunizieren wenn nicht möglich
```

---

## Phase 5: Projektabschluss

```
[ ] Finales Ergebnis liefern
[ ] Abnahme-E-Mail anfordern: "Hiermit bestätige ich die Abnahme..."
[ ] Schlussrechnung stellen
[ ] Zahlung abwarten
[ ] Nach Zahlung: Projektdateien / Zugänge übergeben
[ ] Passwörter ändern / Zugänge entfernen die du nicht mehr brauchst
[ ] Backup aller Projektdateien (2 Kopien!)
[ ] Kunden um Testimonial bitten!
[ ] Kunden nach Weiterempfehlung fragen
[ ] Kunde für Portfolio-Nutzung bestätigen lassen
```

---

## Phase 6: Nachbereitung

```
[ ] Projektdaten ins Buchhaltungssystem eintragen
[ ] Belege ablegen
[ ] Stunden dokumentieren (für nächste Kalkulation!)
[ ] Was lief gut? Was könnte besser sein? (Kurze Notiz)
[ ] Retainer anbieten: "Möchten Sie regelmäßige Betreuung?"
```
