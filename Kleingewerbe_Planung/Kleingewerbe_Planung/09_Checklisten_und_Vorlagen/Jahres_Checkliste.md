# Jahres-Checkliste – Was du jedes Jahr erledigen musst

---

## Januar

```
[ ] Jahresrückblick: Umsatz, Gewinn, Kunden des Vorjahres
[ ] Ziele für das neue Jahr setzen
[ ] Preise überprüfen → jährlich 10-15% erhöhen!
[ ] Versicherungen prüfen (Haftpflicht noch ausreichend?)
[ ] Buchhaltung Vorjahr abschließen und bereit für Steuerberater
[ ] Lohnsteuerbescheinigung von Amazon abholen/prüfen
```

---

## Februar / März

```
[ ] Steuerberatungstermin vereinbaren
[ ] Alle Belege des Vorjahres vollständig?
[ ] EÜR für Vorjahr erstellen / prüfen lassen
[ ] Vorauszahlungen für aktuelles Jahr klären
```

---

## Bis 31. Juli

```
[ ] Einkommensteuererklärung einreichen!
    (Mit Steuerberater: Verlängerung bis Feb. des Folgejahres möglich)
[ ] EÜR beigefügt
[ ] Anlage G (Gewerbe) ausgefügt
[ ] Anlage N (Ausbildungsgehalt) ausgefüllt
```

---

## Laufend (monatlich)

```
[ ] Alle Einnahmen verbucht
[ ] Alle Ausgaben verbucht und Belege abgeheftet
[ ] Rechnungen: Sind alle bezahlt? Mahnungen nötig?
[ ] Steuern zurücklegen (30% des Gewinns aufs Steuerkonto)
[ ] Stundenzettel Danil + Thimofej unterschrieben
[ ] Minijob-Abgaben gezahlt (automatisch per Lastschrift)
[ ] Lohnnachweis an Danil (für sein Jobcenter)
```

---

## Laufend (quartalsweise)

```
[ ] Gewinn-/Verlustrechnung: Wie läuft das Business?
[ ] Umsatz-Voranmeldung (falls Regelbesteuerung)
[ ] Gewerbesteuer-Vorauszahlung (falls fällig): 15. März / 15. Juni / 15. Sept. / 15. Dez.
[ ] Einkommensteuer-Vorauszahlung: 10. März / 10. Juni / 10. Sept. / 10. Dez.
[ ] Equipment-Zustand prüfen: Brauche ich neue Anschaffungen?
```

---

## Jährlich (Ende des Jahres – Dezember)

```
[ ] Große Anschaffungen noch dieses Jahr? (Senkt Jahresgewinn!)
[ ] Rechnungen die noch offen sind abmahnen/abschreiben
[ ] IHK: Beitragsbefreiung erneuern wenn nötig
[ ] Versicherungen: Läuft alles weiter? Preis vergleichen
[ ] KSK (wenn Mitglied): Einkommensmeldung für das nächste Jahr
[ ] Ausrüstung: AfA-Tabelle aktualisieren
[ ] Berufsgenossenschaft: Jahresnachweis einreichen
[ ] Datenschutzerklärung auf Website aktuell?
[ ] AGB auf Website aktuell?
[ ] Impressum auf Website aktuell? (Steuernummer geändert?)
```

---

## Wenn sich was ändert – sofort melden!

| Änderung | Wo melden |
|---|---|
| Adresse ändert sich | Gewerbeamt + Finanzamt + IHK + Berufsgenossenschaft |
| Tätigkeiten ändern sich | Gewerbeamt (Ummeldung) |
| Umsatz überschreitet 25.000 € | Finanzamt → Regelbesteuerung |
| Mitarbeiter einstellen | Bundesagentur für Arbeit (Betriebsnummer) + Minijob-Zentrale |
| Mitarbeiter kündigt | Minijob-Zentrale abmelden |
| Gewerbe aufgeben | Gewerbeamt (Abmeldung) + Finanzamt |
| Danil findet neuen Job | Minijob-Zentrale: Beschäftigung beenden |
| Thimofej wird 18 | Neuen Vertrag, Mindestlohn anpassen |
